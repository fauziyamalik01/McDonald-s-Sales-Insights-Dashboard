## McDonald-s-Sales-Insights-Dashboard
### This is an interactive dashboard for sales insights of McDonald's. It contains various visuals with interactive slicers in order to check the insights.

## Goal of the Project
Create an Interacitve & Dynamic Dashboard to analyze various aspects of sales performance, item popularity, customer preferneces and time-based trends.

## About the datasets
We were provided with 2 seperate files:
1. menu.xlsx (It contains menu_item_id, Item_name, category and price)
2. orders.csv (it contains order_id, order_details_id, order_date and order_time)

## Extraction, transformation and Loading
The entire process of ETL(Extraction, transformation and Loading) is done on Power Query Tool of Advanced Excel as it is one of the major tool of Excel which is used for data transformation, cleaning and even for automation.

## Driven Insights
1. Burgers contribute highest revenue, highlighting strong demand
2. Weekends yields more revenue than weekdays which urges the need for discounts on weekdays
3. Count of orders and Total revenue rises in the Afternoon and Evening which is a great indication for promotional campaigns
